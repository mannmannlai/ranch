Theme Installation

After Installing a fresh wordpress go to Appearence > Themes and then click on Add New. Now click on Upload Theme. Now theme zip file (genz.zip) can be uploaded here. After successfully installing theme Genz theme will display in Appearence > Themes. Now click on Activate. Theme is activated successfully. Theme installation screenshots are given below.

Plugin setup

After activation Genz theme you can see plugin installation message in your admin dashboard. You need to install and active all theme required plugins to achieve better experience of the them.

Install and activate all plugins

Demo Import

After activation Genz theme and successfully activating all theme required plugins the system is ready to import demo data. To import demo data One Click Demo Import plugin must be activated. One Click Demo Import plugin is given as theme required plugin here. To import demo data go to Appearence > Import Demo Data and click on Import Demo Data as marked in the screenshot below.

Customizer settings

Theme template modifications can be made customizer settings. Settings can be accessed quickly by clicking on the pencil icon marked red in screenshot below. Customizer settings are :

  • Typography settings
  • Header settings
  • Footer settings
Typograpy settings

Go to : Appearence > Customize > Design options

Global and heading typography can be customized here. Font family, colors and font sizes can be customized easily.

Header settings

Go to : Appearence > Customize > Theme Options > Header options

Navbar search form and button can be customized here.

Footer settings

Go to : Appearence > Customize > Theme options > Footer

Footer widget area, copyright text can be controlled easily

Post Taxonomies

Post type "post" has Category and tag as default taxonomy. The theme has feature to add custom image to category and tag. Posts under a category can be displayed in five diferent styles.

Newsletter Setup

To setup Newsletter email subscription form "Control Email Subscriber" plugin must be installed and activated. Please go to Settings > Email Subscription and setup you form data.

Newsletter form can be used inside About Us elementor widget. It can be used as a shortcode too. Use this shortcode to display newsletter form [control_email_subscriber_form]
